Board settings

Enter a number between 0 and 1. To disable this MOD, enter 0. To ensure that Waldo appears on every page, enter 1.

The image of Waldo can be made into a hyperlink, so that clicking on it leads the user to another page. To enable this, enter the URL here, include the http:// if it's an offsite link); to disable, leave this field blank.

Set the vertical and horizontal parameters of the area in which Waldo can appear on a page. Waldo can appear anywhere between the left-hand side of the window and the horizontal maximum (as defined here) horizontally, and the top of the page and the vertical maximum (as defined here) vertically. Use the default parameters if confused.
Ultimate Points MOD extension

The number of points to be awarded to each user who stumbles upon Waldo. Set to 0 to disable. Only registered users will be able to receive points.
